# Soluble Immune Factor Profiles in Blood and CSF Associated with LRRK2 Mutations and Parkinson's Disease

Project Overview: Included in this repository is the R code used to create figures and analyze LCC data as well as the Python code used to create GPT-4o prompts and figures. The data used here is from LRRK2 Cohort Consortium serum samples and can be found in the MJFF/LCC Database (www.michaeljfox.org
/data-sets; RRID:SCR_020044).

Acknowledgements: This research was funded in part by Aligning Science Across Parkinson's ASAP-000312 through the Michael J. Fox Foundation for Parkinson's Research (MJFF) and NIH grant R01NS102735.
